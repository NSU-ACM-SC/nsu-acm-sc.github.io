Eligibility Criteria

  • Must be a current, active NSU ACM Student Chapter Member
  • Proficiency in at least one basic programming language — C/C++, Java, Python, or C#
  • Familiar with core Git commands (clone, push, pull, commit, branch, merge)
  • Experience with GitHub repository collaboration (pull requests, code reviews, branching workflow)
  • Ability to Vibe Code — leveraging Prompt Engineering to work alongside AI tools (e.g., ChatGPT, Claude, Copilot) to accelerate development, debug smarter, and ship faster

Job Responsibilities

  • Building and structuring web pages — Implement the HTML structure for all assigned pages (Landing, HackNSU, IUPC, Gaming, Hour of Code, and About Us) following the site architecture and navigation layout defined in the SRS.
  • Responsive UI development — Write clean, mobile-first CSS using Tailwind to ensure the website renders consistently across all screen sizes from 320px to 1920px, covering desktop, tablet, and mobile devices.
  • Integrating content and assets — Embed provided content such as event schedules, prize breakdowns, team photos, sponsor logos, and external links (PDFs, Google Forms) into the appropriate page sections once approved by the Project Lead.
  • Implementing animations and visual effects — Apply lightweight scroll animations (e.g., using AOS or Framer Motion) for elements like the hero banner, prize sections, and event cards, while ensuring performance stays within the 3-second page load requirement.
  • Testing and bug fixing — Cross-browser test all pages on Chrome, Firefox, Safari, and Edge; report and resolve UI inconsistencies; and support the team during the UAT phase to ensure the site is stable and ready for launch.
